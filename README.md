# Wazuh Lab: Failed Login Simulation

This repository contains a lab simulation where I generated failed login attempts on an Ubuntu agent monitored by Wazuh. The event was captured in **Discover**.

- **Agent:** ubuntu-agent
- **Event type:** Failed login
- **Source IP:** 192.168.1.150
- **Severity:** Level 7
